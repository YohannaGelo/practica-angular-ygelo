Acerca de

¿Por qué esta aplicación?

Esta aplicación fue creada para gestionar componentes de ordenadores, permitiendo realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) a través de una API en PHP. El objetivo es demostrar las habilidades adquiridas en el desarrollo de aplicaciones web con Angular.
